# Special Search Filters

Type a filter in the search bar to filter entries by data type.

| Search | Matches |
|--------|---------|
| `:image` | jpg, jpeg, png, webp |
| `:md` | md |
| `:tex` | tex |
| `:scripta` | scripta |
| `:pdf` | pdf |
| `:mp3` | mp3 |
| `:json` | json |
| `:html` | html |
| `:txt` | txt |
| `:binary` | pdf, jpg, jpeg, png, webp, mp3 |

Any other text searches by key or description (case-insensitive).
